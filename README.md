# Personal Website - daymoon

A modern personal website 

## Tech Stack

- **Framework:** Next.js 14 with TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui + Radix UI
- **Icons:** React Icons
- **Animation:** Framer Motion
